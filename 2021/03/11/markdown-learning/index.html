插入代码的几种方式：

  1. 如果是段落上的一个函数或片段的代码可以用反引号把它包起来，例如：
    print() 函数

  2. 用四个空格或一个制表符（tab）直接插入代码块（为什么这里渲染失败了）：
    代码块

  3. 用三个连续的反引号包裹一段代码，并指定一种语言（也可以不指定）：

    $(document).ready(function () {
 alert('RUNOOB');
